# Class Reminder Bot

Telegram + Discord bot that reminds users 5 minutes before their lesson.

## Setup

1. Install Python 3.10+

2. Install dependencies

pip install -r requirements.txt

3. Create `.env` file

Copy `.env.example` and insert your bot tokens.

4. Run

python main.py

## Features

- Telegram bot
- Discord bot
- Shared database
- Reminders 5 minutes before lessons
